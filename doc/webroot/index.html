Topics

The C++ Practicum is a place to explore the more advanced and practical aspects of the C++ programming language. We directly apply abstract software concepts. During the semester students will learn:

  • Generic Programming via Templates
  • Interface Design via Compile-Time/Runtime Polymorphism
  • Performance vs. Programmer Productivity Trade-Offs
  • Unit Testing, and Mock Testing
  • Analysis Tools including gprof, and valgrind.

In addition to technical tools, we also focus on process, and idioms which tame the unyielding power of C++. The semester uses a single project broken down into smaller modules. Each module is managed as an Agile Sprint, affording students a taste of software engineering outside of academia. Specifically, we apply modern C++ design principles including:

  • RAII
  • Smart-Pointers
  • Lambda Expressions
  • Iterators
  • Operator-Overloading
  • Exceptions, esp. Exception Safety
The C++ Practicum is an exciting class, taught by a working software engineer. Please join us for another exciting semester.
